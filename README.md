# VirtualPet - Client Frontend

Este es el repositorio del frontend para los clientes de VirtualPet.

## Tecnologías

- React
- Vite
- Tailwind CSS
- Zustand (Estado)
- Axios (API)

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Construcción

```bash
npm run build
```
